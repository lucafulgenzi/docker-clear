## Usage 
    git clone https://github.com/lucafulgenzi/docker-clerar.git
    cd docker-clear
    bash docker-clear.sh  OR  zsh docker-clear.sh
    
## Create alias
If you use bash edit .bash_profile

If you use zhs edit .zshrc
    
    nano ~/.zshrc
    alias docker-clear="zsh docker-clear.sh"
    source ~/.zshrc
